# Read GPS example
This code example showcase the main functionality of the GPS library, namely how to read and process the GPS data. 
It is almost self-contained from the other OLB libraries, save for the SD card library which it calls upon to log measurements 
as they are taken. The script can be tuned with the ```config.h``` file that can be found in the ```src``` directory. An exhaustive list of paramters is given below
